# GovCheck

> SPDX-License-Identifier: AGPL-3.0-or-later

## Overview

GovCheck is a modern, accessible web interface designed to validate compliance for research proposals (Federal Compliance/FAR). It provides an intuitive environment for uploading documents and viewing AI-generated compliance analysis results.

**Note**: This is the **Frontend Application** (Next.js/React). It can run in a standalone "mock mode" for demonstration and UI development without a live backend connection.

## Development

This project is built with:
- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Testing**: Vitest & Playwright

### Key Commands

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start development server |
| `pnpm build` | Build for production |
| `pnpm test` | Run unit tests |
| `pnpm lint` | Run code linting |
